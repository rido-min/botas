Implement TurnState across .NET, Node, and Python (#361)

Adds turn-scoped state persistence with three scopes (conversation,
user, temp), MemoryStorage and FileStorage adapters, and opt-in
middleware integration (`app.UseState(storage)` / useState / use_state).

Architecture decisions (Decision A6, locked by Rido):
- Middleware-based integration, not built into BotApplication core
- Atomic per-turn semantics: state writes are discarded on handler error
- v1 ships MemoryStorage + FileStorage; cloud adapters deferred

Cross-language parity:
- Spec defines a canonical FileStorage key encoding (RFC 3986
  percent-encoding via Uri.EscapeDataString / encodeURIComponent /
  urllib.parse.quote(safe="")) so the same key produces the same
  filename across languages.
- Cross-language parity tests guard the encoding contract.
