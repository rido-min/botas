Add 06-state-bot samples for .NET, Node, and Python (#361)

Adds runnable counter-bot samples demonstrating TurnState with FileStorage
across all three scopes (conversation, user, temp). Sample build also
surfaced and closed an API gap: BotApp hosting wrapper now exposes
UseState() / useState() in both .NET (Botas) and Node (botas-express),
mirroring how the underlying BotApplication already supported it.

Each sample:
- Uses FileStorage with ./state-data/ (visible JSON files for inspection)
- Increments turn_count (conversation scope) and user_message_count (user scope)
- Uses temp scope for per-turn formatted reply
- Supports "reset" (clear conversation) and "whoami" (show user state)
