Table of Contents

Namespace Botas

Classes

ActivityType

Core activity type string constants used by BotApplication for dispatch.

AppBuilderExtensions

Extension methods for configuring a BotApplication on an ASP.NET Core pipeline. Registers authentication, authorization, exception handling, and the messages endpoint.

Attachment

Represents a file or rich card attachment on a Bot Service activity.

BotApp

Zero-boilerplate bot host that wraps ASP.NET Core setup.

var app = BotApp.Create(args);
app.On("message", async (ctx, ct) =>
    await ctx.SendAsync($"Echo: {ctx.Activity.Text}", ct));
app.Run();

BotApplication

Core bot application that processes incoming activities through a middleware pipeline and dispatches them to registered handlers. This is the central entry point for the Botas SDK.

BotApplicationConfigurationExtensions

Extension methods for registering BotApplication and its dependencies (authentication, authorization, conversation HTTP client, token acquisition) in the DI container.

BotHandlerException

Exception thrown when an activity handler or invoke handler raises an unhandled error. Wraps the original exception and includes the Activity that caused the failure.

CardAction

Represents an actionable button that can be presented to the user. Used within SuggestedActions or as part of card content.

ChannelAccount

Identifies a user, bot, or other participant in a conversation.

ChannelInfo

Information about a Microsoft Teams channel.

Conversation

Identifies a conversation within a channel. Contains the conversation ID and any channel-specific extension properties.

ConversationClient

HTTP client for sending outbound activities to the Bot Service channel service. Handles SSRF protection by validating service URLs against a known allowlist.

CoreActivity

Represents a Bot Service activity — the fundamental unit of communication between a bot and a channel. Contains typed fields for common properties and an extension dictionary that preserves unknown JSON properties.

CoreActivityBuilder

Fluent builder for constructing outbound CoreActivity instances.

Entity

Represents a semantic entity (mention, place, thing, etc.) attached to an activity. The Type discriminates the entity kind; additional data is captured in Properties.

InvokeResponse

Response returned by an invoke activity handler. The Status is written as the HTTP status code; Body is serialized as JSON.

JwtExtensions

Extension methods for configuring JWT-based Bot Service authentication and authorization. Supports both single-tenant and multi-tenant bot registration scenarios.

MeetingInfo

Information about a Microsoft Teams meeting.

NotificationInfo

Notification settings for a Teams activity.

RemoveMentionMiddleware

Middleware that strips @mention text from the incoming activity when the mention refers to the current bot (matched by AppId). After removal the activity text is trimmed of leading/trailing whitespace.

SuggestedActions

A set of suggested actions that the channel can present to the user as quick-reply buttons.

TeamInfo

Information about a Microsoft Teams team.

TeamsActivity

Teams-specific activity with strongly-typed channel data and helper methods.

TeamsActivityBuilder

Fluent builder for constructing outbound TeamsActivity instances.

TeamsActivityType

Extended activity type string constants for Teams and other Bot Service channels. Includes all core types plus channel-specific activity types.

TeamsChannelAccount

A ChannelAccount extended with Microsoft Teams–specific properties such as email address and user principal name.

TeamsChannelData

Teams-specific channel data containing tenant, channel, team, meeting, and notification information. Deserialized from the channelData property of a Teams activity.

TeamsConversation

A Teams-specific conversation that extends Conversation with Teams metadata such as conversation type, tenant ID, and group indicator.

TenantInfo

Microsoft Teams tenant information from channel data.

TurnContext

Context for a single activity turn, passed to handlers and middleware. Provides the incoming activity, a reference to the bot application, and a scoped SendAsync(string, CancellationToken) method that automatically routes replies back to the originating conversation.

Interfaces

ITurnMiddleWare

Interface for middleware components that participate in the bot turn pipeline. Middleware is invoked in registration order before the activity handler.

Delegates

NextDelegate

Delegate that advances the middleware pipeline to the next middleware or the final handler.
