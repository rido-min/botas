AddBotAuthentication(IServiceCollection, string)

Adds JWT bearer authentication for a single-tenant bot registration. Configures two authentication schemes: "Bot" (for Bot Service tokens) and "Agent" (for agent-to-agent tokens from the configured tenant).

public static AuthenticationBuilder AddBotAuthentication(this IServiceCollection services, string aadSectionName = "AzureAd")

Adds a JWT bearer authentication scheme configured for Bot Service token validation. Dynamically resolves the OIDC metadata endpoint based on the token's issuer claim.
