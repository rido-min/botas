# Advanced Hosting — Flask

**Category:** 2 — Advanced Hosting  
**Language:** Python  
**Complexity:** Intermediate

## What This Sample Demonstrates

- Using botas with **Flask** for traditional web server control
- Manual auth middleware integration with Flask
- That botas is web server agnostic — works with any Python framework

## Prerequisites

- Python 3.11+
- No Azure credentials needed for local testing

## Run

```bash
pip install -e .
python main.py
```

- `main.py` — Flask server with botas integration
