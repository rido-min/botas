# Echo Bot

**Category:** 1 — Basic Bot
**Language:** Node.js
**Complexity:** Basic

## What This Sample Demonstrates

- The simplest possible bot using `BotApp`
- Echoing user messages back

## Prerequisites

- Node.js 20+
- No Azure credentials needed for local testing

## Run

```bash
npx tsx index.ts
```
