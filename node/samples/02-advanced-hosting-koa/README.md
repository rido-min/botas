# Advanced Hosting — Koa

**Category:** 2 — Advanced Hosting  
**Language:** Node.js  
**Complexity:** Intermediate

## What This Sample Demonstrates

- Using botas-core with **Koa** for full web server control
- Custom auth middleware pattern for Koa
- That botas-core is web server agnostic — works with any Node.js framework

## Prerequisites

- Node.js 20+
- No Azure credentials needed for local testing

## Run

```bash
npx tsx index.ts
```

## Key Files

- `index.ts` — Koa server with botas-core integration
