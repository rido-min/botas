# Advanced Hosting — Hono

**Category:** 2 — Advanced Hosting  
**Language:** Node.js  
**Complexity:** Intermediate

## What This Sample Demonstrates

- Using botas-core with **Hono** for full web server control
- Custom auth middleware pattern for Hono
- That botas-core is web server agnostic — works with any Node.js framework

## Prerequisites

- Node.js 20+
- No Azure credentials needed for local testing

## Run

```bash
npx tsx index.ts
```

- `index.ts` — Hono server with botas-core integration
